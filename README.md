# 🔮 OracleLuck

> 东方文化 × AI 对话平台 | AI-powered Eastern cultural exploration platform

[![Website](https://img.shields.io/badge/Web-oracleluck.com-gold)](https://oracleluck.com)

OracleLuck 是一个融合东方玄学解读与 AI 对话的双语文化平台。

## ✨ Features

- 📖 **I Ching 64 Hexagrams** — AI-powered interpretation of the 3,000-year-old classic
- 🎭 **30+ Deity Chats** — Chat with Guanyin, Guan Gong, Athena, Anubis and more
- 🃏 **Tarot 78 Cards** — AI readings of Major & Minor Arcana
- 🌍 **Bilingual** — Chinese (Simplified & Traditional) + English

## 🛠 Tech Stack

- **Frontend**: Next.js 15 App Router + React + TypeScript + Tailwind CSS
- **AI Engine**: DeepSeek API
- **Database**: Supabase PostgreSQL
- **Deployment**: Nginx + PM2 on Ubuntu
